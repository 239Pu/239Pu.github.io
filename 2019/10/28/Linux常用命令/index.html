Linux 常用命令

常用命令

命令 说明
ls 查看当前目录文件
cd 目录操作
touch 创建文件
mkdir 创建文件
cp 复制操作
mv 剪切操作
cat 查看文件内容
more 从头显示一页内容
head 从头显示 默认显示10行 可以控制显示行数”head -n 行数 文件”
tail 从尾显示 默认显示10行 可以控制显示行数”tail -n 行数 文件”
find 查找文件 “find -name 文件或目录名”
grep 查找文件中的字符 “grep “查找内容” 路径”
whereis 查找某个命令所在位置
which 查找当前用户名的命令
awk sed 两个命令需要自学
ctrl + c 强制中断命令
ctrl + z 将任务中断放入后台
jobs 显示当前暂停的进程(与ctrl z配合)
nohup 命令 & 保持后台运行 并输出结果到nohup.out
screen -list 查看当前有几个screen
screen -r id 通过id还原screen
tar cvf 打tar包 tar cvf 压缩包名.tar 内容1 内容2 ….
tar xvf 解tar包 tar xvf 压缩包名.tar
tar czvf 打tar包 tar czvf 压缩包名.tar.gz 内容1 内容2 ….
tar xzvf 解tar包 tar xzvf 压缩包名.tar.gz
zip -r 压缩 zip -r 压缩包名.zip 内容1 内容2
unzip 解压 unzip 压缩包名.zip
exit 退出登录 推出终端
ctrl + L 清屏
clear 清屏
date 时间
vim 编辑文件
:set nu 显示行号
:set nonu 不显示行号
!wq 强行写
:%s/原文/替换 替换
netstat 查看网络状态
ps 查看运行的程序
top 实时查看运行的程序
## Ubuntu CentOS 软件安装命令
debin特有的(ubuntu/kali)
apt 从软件源安装
apt update 更新软件源
apt install 软件
apt search xxx 搜索软件
apt remove 删除 
centos(red-hat)
yum 从软件源安装
yum update 更新软件源
yum install 软件 -f参数解决依赖
yum search xxx 搜索软件
yum upgrade
yum remove 删除

deb安装包 使用 dpkg -i 包名 安装

Linux用户权限/组命令

useradd 添加用户 直接创建,没有设置
adduser 添加用户 交互式设置密码,用户信息
passwd 修改用户密码
su 切换用户
userdel 删除用户
usermod 修改用户信息 
-L 锁定用户 但是高权限可用su 登录该用户
-U 解除锁定 原理:在密码位置添加!,使用户不能密码登录
-g 修改指定用户群组
-G 修改附加用户群组

/bin/false 不能登录没有回显
/usr/sbin/onlogin //不能登录 有回显 提示账号不能登录
常用shell:
/bin/bash
/bin/sh
/bin/tsh
/bin/ksh
-u 修改uid 只能选用未使用的id号
vim /etc/passwd修改uid可以改为任意uid
linux系统通过uid识别用户,相同的uid,视为相同的用户
id命令 显示当前用户uid gid 所用组织的用户/用户信息
/etc/passwd 里边记录的当前系统所有用户,他记录了关于用户登录信息 uid gid 使用主目录 shell等
/etc/shadow 里边记录用户的密码,生存周期,$1 $5 $6, 重点:!,在前会禁止用户密码登录
/etc/group 里边记录
groupadd 新建用户组
groupdel 删除用户组
groupmod 修改用户所属组
ssh 开root登录  /etc/ssh/sshd_config 
#PermitRootLogin prohibit-password
修改为PermitRootLogin yes

ssh登录
1密码登录
2密钥登陆
1.生成密钥
cd ~/.ssh (当前用户)
ssh-keygen -t RSA 默认生成 
当前用户的ssh_key复制id_rsa.pub到authorized_keys
将ssh_key传输到电脑上重启ssh服务
2使用密钥登录打开xshell新建连接身份认证处先则使用public key登录 加载ssh_key
chmod #文件权限操作
r =4 #读
w =2 #写
x =1 #执行
7=r+w+x
1,chmod 777 filemode
2,chmod u/g/o/a [+/-/=] r/w/x filename

chown修改当前文件或文件夹所属的用户
chown -R root:root filename

Git仓库

git 
1.我们使用了当前的用户以及密码去克隆仓库,是什么权限,那麽这个用户是否可以登录系统
git加固 -禁止git用户登录:设置/etc/passwd 用于连接GIT的用户, shell为/usr/bin/git-shell
2.使用其他可以登录的用户是否可以clone仓库
看仓库权限,有权限就可以克隆,比如root
初始化仓库
git init --bare xxxx.git
克隆仓库
git clone ssh://user@ip:port/路径
