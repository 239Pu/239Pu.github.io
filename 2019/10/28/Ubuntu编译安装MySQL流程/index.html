Ubuntu编译安装MySQL流程

发表于 2019-10-28 分类于 编译安装

安装依赖软件

apt-get install make cmake gcc g++ bison libncurses5-dev libssl-dev

创建MySql目录

mkdir -p /usr/local/mysql
mkdir -p /usr/local/mysql/data
mkdir /usr/local/mysql/{logs,pids}

上传MySql源码压缩包 并解压

例如: /root/mysql-5.6.46.tar.gz
解压 tar xzvf mysql-5.6.46.tar.gz
cd ./mysql-5.6.46.tar.gz

cmake 修改安装配置

cmake -DCMAKE_INSTALL_PREFIX=/usr/local/mysql \
-DMYSQL_DATADIR=/usr/local/mysql/data \
-DSYSCONFDIR=/etc \
-DWITH_MYISAM_STORAGE_ENGINE=1 \
-DWITH_INNOBASE_STORAGE_ENGINE=1 \
-DWITH_MEMORY_STORAGE_ENGINE=1 \
-DWITH_READLINE=1 \
-DMYSQL_UNIX_ADDR=/var/lib/mysql/mysql.sock \
-DMYSQL_TCP_PORT=3306 \
-DENABLED_LOCAL_INFILE=1 \
-DWITH_PARTITION_STORAGE_ENGINE=1 \
-DEXTRA_CHARSETS=all \
-DDEFAULT_CHARSET=utf8 \
-DDOWNLOAD_BOOST=1 \
-DWITH_BOOST=/root/mysql-5.6.46/boost \
-DDEFAULT_COLLATION=utf8_general_ci

运行完cmake 开始编译

make -j 线程数
make install

配置环境变量

vim /etc/profile
尾部添加
export PATH=/usr/local/mysql/bin:/usr/local/mysql/lib:$PATH
使文件生效
source /etc/profile

mysqld配置

mysqld --initialize-insecure --user=mysql --basedir=/usr/local/mysql --datadir=/usr/local/mysql/data

配置开机启动

cd /usr/local/mysql
cp support-files/mysql.server /etc/init.d/mysqld
chmod a+x /etc/init.d/mysqld
systemctl enable mysqld.service

编辑MySql配置文件

vim /etc/my.cnf
修改为
[mysqld]
datadir=/usr/local/mysql/data
socket=/usr/local/mysql/mysql.sock
[mysqld_safe]
log-error=/usr/local/mysql/logs/mysqld.log
pid-file=/usr/local/mysql/pids/mysqld.pid
symbolic-links=0
[client]
default-character-set=utf8
socket=/usr/local/mysql/mysql.sock
[mysql]
default-character-set=utf8
socket=/usr/local/mysql/mysql.sock
!includedir /etc/my.cnf.d
保存退出

创建用户 并给与文件和文件夹权限

groupadd mysql
useradd -r -g mysql mysql
mkdir -p /var/lib/mysql/
mkdir -p /etc/my.cnf.d/
touch /usr/local/mysql/logs/mysqld.log
touch /usr/local/mysql/pids/mysqld.pid
chown mysql:mysql -R /usr/local/mysql/

初始化MySql脚本

/usr/local/mysql/scripts/mysql_install_db --user=mysql --basedir=/usr/local/mysql --datadir=/usr/local/mysql/data

启动服务

service mysqld start #启动服务
service mysqld status
无密码的状态可以直接以mysql命令进入mysql的交互界面
mysql -u root
设置用户和密码
mysqladmin -u root password "root"
使用密码登录
mysql -u root –p
完成
